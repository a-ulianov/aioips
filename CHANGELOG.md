# Changelog

Все значимые изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект следует [семантическому версионированию](https://semver.org/lang/ru/).

## [1.0.0] — 2026-06-26

Первый стабильный публичный релиз.

### Добавлено
- Асинхронный клиент `IPSClient` поверх IPS Server Web API (aiohttp + pydantic v2).
- **723 метода в 42 разделах** — практически полная прикладная поверхность API: чтение и
  запись объектов, связей и файлов, жизненный цикл, состав, версии, права доступа,
  метамодель, документы и редактор документов, проекты, справочная система, ЭЦП,
  снимки, отчёты, конфигурация и др.
- Ядро: два способа авторизации (логин/пароль с авто-refresh JWT или готовый access-токен),
  повторы транзиентных ошибок (`tenacity`), иерархия исключений по HTTP-кодам, поддержка
  `multipart` и потоковых загрузок, хуки наблюдаемости (`MetricsHook`).
- Типизированные pydantic-схемы запросов и ответов с маппингом `snake_case` ↔ `camelCase`.
- Гейт `confirm=True` на разрушающих операциях.
- MCP-grade docstrings на русском для всех публичных методов и схем.

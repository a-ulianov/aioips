# Начало работы

Этот раздел проведёт вас от пустого окружения до первого рабочего запроса к IPS Web API.
Если вы видите `aioips` впервые — читайте страницы по порядку: каждая опирается на предыдущую.

`aioips` — это асинхронный Python-клиент для **IPS Server Web API** (Intermech Professional
Solutions). Он скрывает за чистым типизированным интерфейсом всё «транспортное»: JWT-авторизацию
с автоматическим обновлением токена, повторы при сбоях сети и разбор ответов сервера. Вы пишете
бизнес-логику, а не возитесь с заголовками и токенами.

## Что вам понадобится

- **Python 3.13+** и доступ в интернет для установки зависимостей.
- **Адрес сервера IPS Web API** (вида `http://your-ips-host:8080`).
- **Учётные данные** IPS (логин и пароль) — либо заранее полученный JWT access-токен.

!!! warning "Секреты — только в окружении"
    Реальные адреса серверов, логины и пароли держите в переменных окружения или в файле
    `.env` (он добавлен в `.gitignore`). Никогда не вписывайте их прямо в код и не коммитьте
    в репозиторий.

## Самый короткий путь

Если хочется сразу увидеть код в действии:

1. [Установите](installation.md) пакет.
2. Задайте параметры подключения через [`.env`](configuration.md).
3. Запустите пример из [Быстрого старта](quickstart.md).

# aioips

**Асинхронный Python-клиент для IPS Server Web API** (Intermech Professional Solutions).

`aioips` даёт удобный `async/await`-интерфейс к REST API системы IPS: авторизацию по JWT с
автоматическим обновлением токена, типизированные `pydantic`-схемы запросов и ответов, повторы
при транзиентных ошибках и понятную иерархию исключений. Клиент покрывает **723 метода** в
**42 разделах** API — практически всю прикладную поверхность чтения плюс запись объектов, связей
и файлов.

## Что это и зачем

IPS — это PDM/PLM-система управления инженерными данными (изделия, документы, состав, маршруты).
Её сервер предоставляет REST Web API, но «сырой» API многословен: JWT-авторизация, двухуровневая
идентичность «объект/версия», обёртки результатов, особенности кодировки. `aioips` скрывает эту
сложность за чистым типизированным интерфейсом, чтобы вы писали бизнес-логику, а не транспорт.

```python
import asyncio
from aioips import IPSClient, IPSConfig


async def main() -> None:
    config = IPSConfig(
        base_url="http://your-ips-host:8080",
        login_name="your-login",
        password="...",          # лучше — через переменную окружения IPS_PASSWORD
        role_name="Администратор",
    )
    async with IPSClient(config=config) as ips:
        me = await ips.user_info()
        print(me.login_name, me.is_admin)

        # все типы объектов метамодели
        types = await ips.object_types()
        print("Типов объектов:", len(types))


asyncio.run(main())
```

## Ключевые возможности

- **Async-first** на базе `aiohttp` — параллельные запросы без блокировок.
- **JWT-авторизация** скрыта: токен получается лениво и обновляется автоматически (в т. ч. по `401`).
- **Типизированные схемы** (`pydantic` v2) с маппингом `snake_case` ↔ `camelCase`.
- **Повторы транзиентных ошибок** (5xx, 429, сетевые) с экспоненциальной задержкой.
- **Безопасность записи**: разрушающие операции защищены гейтом `confirm=True`.
- **MCP-grade docstrings**: каждый метод самодостаточно описывает, *что* он делает, *когда* и *как*
  его применять — пригодно и для людей, и для AI-агентов.
